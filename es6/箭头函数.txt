
箭头函数 arrow function  ES6新定义的函数

  eg：x => x*x;(单行) 相当于 function(x){return x*x;}

  eg:x =>{
       if(x>0){return x*x};
       if(x<=0){return -x*x}；	   
  }(多行)
  
  eg: (x,y,z...rest) => {
       var i,sum = x + y;
	   for(i=0;i<rest.length;i++){
	     sum += rest[i];
	   }
	   return sum;
  };(多参)
  
  eg: x => ( {foo:x});(返回的是对象时)
  
  箭头函数中的this总是指向词法作用域 也就是外层调用者obj，因此用call() 和 apply()调用箭头函数时，无法绑定this.传入其中的第一个参数会被忽略。
