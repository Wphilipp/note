
一、变量作用域

   命名空间
   为了避免命名冲突，通常创建一个全局变量，把自己的所有变量和函数都绑定到一个全局变量中
   
   eg:var myAPP = {}；
          myAPP.name = 'zhiHu';
		  myAPP.read = function(){};
		  
	局部作用域
	ES6中引入了新关键词 let，let可以声明一个块级作用域变量；
	 
	  'use strict'
	  function foo (){
	   var sum = 0;
	   for(let i=0;i<10;i++){
	      sum +=i;
	   }
	   alert(i)//referenceError
	  }
	  
	  常量
	  在ES6之前，通常通过全部大写的变量来表示这个是一个常量，不要修改它的值；
	  ES6中新加了const关键词，用来声明一个常量；const和let都具有块级作用域；
	  
	  eg:  const A = 3;
